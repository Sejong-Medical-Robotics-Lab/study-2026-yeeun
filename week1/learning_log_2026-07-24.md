## 직접 해본 것 / 성공한 것
- pwd, ls, cd, mkdir 명령어들을 직접 입력해보며 작업공간 폴더를 생성하고 자유롭게 이동해보았습니다.
- 3장 내용을 토대로 터미널 명령어 하나로 매일 날짜별 학습일지 파일을 자동 생성하는 쉘 스크립트를 작성하고 실행에 성공했습니다.
- 개인 학습repo를 클론하고, 수정 사항을 명령어로 올린 후 깃허브에 푸시하는데 성공했습니다.
- 폴더나 파일 명에 오타가 생겼을 때 mv 명령어를 이용해 터미널에서 직접 수정을 해보았습니다.

## 막힌 것 / 이해가 안 되는 것 (다음 Q&A 질문할 목록)
- 학습일지 스크립트를 진행할 때 경로가 week1/code 내부로 고정되거나 에러가 발생해 실행되지 않았었습니다.
현재 터미널 위치를 pwd로 정확히 확인하고 올바른 Git 저장소 폴더로 이동한 뒤, 스크립트 파일에 실행권한(chmod +x log.sh)을 부여하였더니 정상적으로 실행되었습니다.
- 작업 중인 week1 디렉토리 안에 아직 아무런 파일도 생성해두지 않아 경로가 꼬이는 상황이 생겼었습니다.
파일이 없는 초기 상태에서 발생한 경로 특성을 이해하고, 현재 위치를 정확히 확인한 뒤 필요한 파일과 스크립트를 생성하여 해결하였습니다.

## 한줄 소감

하지만 반복해서 복습하고 여러 미션들을 직접 수행해 나가다 보니 명령어들이 점차 손에 익기 시작했고, 처음보다는 속도가 빨라진 거 같아 성취감을 작게나마 느낄 수 있었습니다.
아직은 기초 단계이지만 앞으로 해볼 다양한 실습 활동들이 더욱 기대되는 것 같습니다.
